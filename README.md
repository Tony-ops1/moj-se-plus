# ⚡ MOJ SE Plus

HTML kalkulator za sončno elektrarno, električno vozilo in povračilo investicije.

## Datoteke

- `index.html` — aplikacija

## Odpri v brskalniku

Klikni `index.html`, nato `Raw` ali odpri projekt preko GitHub Pages.

## Codex navodilo

Ko odpreš projekt v Codexu, uporabi to navodilo:

```text
Ne spreminjaj izgleda, besedil, postavitve ali animacij. Popravljaj samo JavaScript formule, če najdeš napako. Aplikacija mora ostati ena HTML datoteka.
```

## GitHub Pages

1. Odpri Settings.
2. Klikni Pages.
3. Source: Deploy from a branch.
4. Branch: main.
5. Folder: /root.
6. Save.
